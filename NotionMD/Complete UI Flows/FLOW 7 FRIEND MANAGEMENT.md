# FLOW 7: FRIEND MANAGEMENT

```markdown
Groups Tab → "+" button → "New Group"
  ↓
Create Group Sheet
  ├─ Group name: "College Friends"
  ├─ Add members:
      └─ Search by phone/email
      └─ Import from contacts
      └─ Send invite link
  ↓
Tap "Create Group"
  ↓
Group created, invites sent
  ↓
Recipients get notification: "[Your name] added you to College Friends"
  ↓
They accept → join group → can see shared calendar availability
```

Adding individual friends:

```markdown
Profile Tab → "Friends" → "Add Friend"
  ├─ Search by username/email/phone
  ├─ Send friend request
  └─ OR: Share your friend code/QR code
  ↓
Friend accepts
  ↓
Can now add them to groups
```

# FLOW 8: CALENDAR VIEWS & NAVIGATION

```markdown
Calendar Tab (default: Week View)
  ↓
Swipe gestures:
  ├─ Swipe left/right: Next/previous week
  ├─ Swipe up: Expand to Month View
  └─ Swipe down: Collapse to Day View
  ↓
Toggle buttons (top):
  ├─ [Day] [Week] [Month] view selector
  └─ Filter button:
      ├─ Show all events
      ├─ Only my events
      ├─ Only group events
      └─ By specific group
  ↓
Color coding:
  ├─ Blue = Your personal events
  ├─ Green = Group events you're attending
  ├─ Gray = Your events marked "busy only" (visible to friends)
  └─ Purple = Pending proposals (in your inbox)
```

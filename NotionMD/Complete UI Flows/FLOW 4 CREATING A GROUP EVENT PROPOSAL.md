# FLOW 4: CREATING A GROUP EVENT PROPOSAL

```markdown
Calendar View OR Groups Tab
  ↓
Tap "+" → "Propose Group Event"
  ↓
Group Event Creation Sheet
  ├─ Event title: "Secret Santa Planning"
  ├─ Select Group: [dropdown of friend groups]
  ├─ Description (optional)
  ├─ Location (optional)
  └─ Propose Time Slots (the key feature)
      └─ "+ Add Time Slot" button
          ├─ Opens time picker
          ├─ User selects: "Dec 15, 6-8pm"
          ├─ Can add multiple slots (2-5 slots)
          └─ Shows which friends are available/busy for each slot
  ↓
Tap "Send Proposal"
  ↓
Notification sent to all group members
  ↓
Event appears in:
  - Your Inbox
  - Recipients' Inbox (with voting UI)
  - Groups tab under that group
```

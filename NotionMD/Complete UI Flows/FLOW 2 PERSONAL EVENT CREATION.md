# FLOW 2: PERSONAL EVENT CREATION

```markdown
Calendar View
  ↓
Tap "+" button (floating action button)
  ↓
New Event Sheet slides up
  ├─ Title field
  ├─ Date/Time pickers
  ├─ Location (optional)
  ├─ Notes (optional)
  ├─ Repeat options
  └─ Privacy Settings ⭐
      ├─ "Private" (default) - only you see it
      ├─ "Shared with name" - friends see "Dinner at Mom's"
      ├─ "Shared - busy only" - friends see "Busy 6-8pm"
      └─ "If shared, who sees it?"
          └─ Select Groups toggle (if not private)
  ↓
Tap "Create"
  ↓
Event appears on calendar
  ↓
Syncs to Apple Calendar (bidirectional)
```

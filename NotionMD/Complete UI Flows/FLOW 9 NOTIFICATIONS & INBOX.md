# FLOW 9: NOTIFICATIONS & INBOX

```markdown
Inbox Tab (badge shows unread count)
  ↓
Categorized list:
  ├─ **Pending Votes** (needs your response)
  │   └─ Event proposal cards (Flow 5)
  ├─ **Confirmed Events** (upcoming group events)
  │   └─ Tap to view details / add to calendar
  ├─ **Friend Requests**
  │   └─ Accept/Decline buttons
  └─ **Activity Feed**
      ├─ "[Friend] joined [Group]"
      ├─ "[Friend] voted on [Event]"
      └─ "Time chosen for [Event]"
  ↓
Pull to refresh
  ↓
Swipe to dismiss/archive
```
